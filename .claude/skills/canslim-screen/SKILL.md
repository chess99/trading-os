# CANSLIM Screen — 基本面选股评分

基于威廉·欧奈尔《笑傲股市》（How to Make Money in Stocks）的 CANSLIM 方法。

欧奈尔的核心洞见：**最大涨幅的股票在起涨前，都有共同的基本面特征。**
技术面告诉你何时买，基本面告诉你该买什么。两者缺一不可。

---

## CANSLIM 七个维度

### C — Current Earnings（当季盈利增长）

**标准**：当季 EPS（每股收益）同比增长 ≥ 25%，且加速增长更好。

分析要点：
- 对比去年同期，增长率是否达标
- 是否连续两季以上加速（从 20% → 30% → 40% 是强信号）
- 剔除一次性收益（资产出售、补贴等）后的核心业绩
- A 股注意：季报数据可能有季节性，同比比环比更可靠

评分：✓（≥25%）/ ⚠（15-25%）/ ✗（<15% 或下滑）

---

### A — Annual Earnings（年度盈利增长）

**标准**：年度 EPS 连续 3 年以上增长，且增速 ≥ 25%。

分析要点：
- 过去 3 年 EPS 是否逐年增长（不能有下滑年）
- 最近一年增速是否仍然维持
- ROE（净资产收益率）≥ 17% 是强信号
- 注意：A 股周期股业绩波动大，需要区分成长股和周期股

评分：✓（3年以上持续增长）/ ⚠（1-2年增长）/ ✗（增长中断或下滑）

---

### N — New（新催化剂）

**标准**：新产品、新服务、新管理层、新行业趋势，或股价突破前期高点。

分析要点：
- 是否有近期的新产品发布或重大业务变化
- 管理层是否有重大调整（通常是正面信号）
- 是否受益于新的行业趋势（AI、新能源等）
- 股价是否接近或突破 52 周新高（突破新高不是追高，是强势信号）

评分：✓（有明确催化剂）/ ⚠（一般）/ ✗（无新故事，靠炒作）

---

### S — Supply and Demand（供求关系）

**标准**：流通股不宜过多；机构持仓比例增加；大股东增持。

分析要点：
- 流通市值：中小盘（50-500亿）比大盘股更容易大涨
- 近一季度机构持仓比例是否增加（基金重仓是正面信号）
- 大股东或高管是否在增持（反向指标：大量减持是警示）
- A 股注意：融资余额增加代表散户加杠杆，需谨慎

评分：✓（机构增持，流通适中）/ ⚠（持仓平稳）/ ✗（机构减持，大股东清仓）

---

### L — Leader or Laggard（行业龙头）

**标准**：行业内相对强度排名前 10-15%，是龙头而非跟风股。

分析要点：
- 同行业中，该股近 6 个月涨幅是否领先同业
- 市场份额是否在扩大
- 品牌和竞争壁垒是否清晰
- 避免买"第二名"：欧奈尔明确说，买龙头，不买跟风

评分：✓（行业前 15%）/ ⚠（中游）/ ✗（行业落后者）

---

### I — Institutional Sponsorship（机构持仓）

**标准**：至少有几家知名机构持仓，且近期持仓增加。

分析要点：
- 公募基金重仓股名单中是否出现
- 近一季度新进机构家数 vs 减持机构家数
- 注意：过度机构化（持仓比例 > 60%）反而是风险（卖出压力大）
- 外资（北向资金）持续流入是强信号

评分：✓（机构增仓）/ ⚠（机构持仓稳定）/ ✗（机构大幅减仓）

---

### M — Market Direction（大盘方向）

**标准**：只在大盘上升趋势中做多，大盘下跌时观望。

分析要点：
- 上证指数 / 沪深 300 是否处于上升趋势（高于 50 日均线）
- 近期是否有"跟进日"（大盘放量大涨，确认反弹有效）
- 市场分布日（大盘放量下跌）是否过多（>5次，熊市信号）
- A 股特殊：政策面是重要变量，需要关注宏观政策导向

评分：✓（牛市，趋势向上）/ ⚠（震荡市，谨慎）/ ✗（熊市，观望）

## 分析流程

### 第一步：获取数据

```bash
# 获取 A 股基本面数据（需要 AKShare）
python -c "
import akshare as ak
# 财务报表
df = ak.stock_financial_report_sina(stock='sh600000', symbol='资产负债表')
print(df.head())
"

# 或通过 trading-os 的数据湖查询已有数据
python -m trading_os query-bars --symbols SSE:600000 --limit 10
```

如果本地没有基本面数据，使用 web_search 搜索：
- `{股票代码} 季报 EPS 增长`
- `{股票代码} 机构持仓 最新`
- `{股票代码} 近期新高`

### 第二步：逐维度评分

对每个维度给出：✓ / ⚠ / ✗，并简要说明数据依据。

### 第三步：综合结论

```
CANSLIM 评分：{股票代码} {股票名称}
═══════════════════════════════════════
C — 当季EPS增长：{数据}  {评分}
A — 年度EPS增长：{数据}  {评分}
N — 新催化剂：{描述}  {评分}
S — 供求关系：{描述}  {评分}
L — 行业地位：{描述}  {评分}
I — 机构持仓：{描述}  {评分}
M — 大盘方向：{描述}  {评分}
───────────────────────────────────────
通过维度：{X}/7
综合评级：强烈关注 / 值得关注 / 一般 / 回避

结论：{2-3句话，说明核心优势和主要风险}

建议下一步：
→ {如果 ≥5/7：运行 elder-screen 确认技术面入场时机}
→ {如果 3-4/7：列出需要改善的维度，等待观察}
→ {如果 ≤2/7：不建议关注，说明主要问题}
```

---

## 评级标准

| 通过维度 | 综合评级 | 建议 |
|---------|---------|------|
| 6-7 / 7 | 强烈关注 | 立即运行技术面分析，准备入场 |
| 4-5 / 7 | 值得关注 | 加入观察名单，等待更多维度改善 |
| 2-3 / 7 | 一般 | 不主动布局，有特殊情况再看 |
| 0-1 / 7 | 回避 | 明确说明问题，不建议关注 |

**M 维度（大盘方向）是否决项**：如果大盘处于明确下跌趋势（熊市），即使其他 6 项全部通过，也不建议入场。等待大盘确认反转。

---

## 注意事项

- CANSLIM 是成长股框架，不适用于银行、地产、公用事业等价值股
- A 股财报数据有滞后，季报发布后才能用，避免使用预测数据
- 这是参考意见，不是投资建议，交易有风险
- 基本面好不代表现在是好的买点，需要配合技术面（elder-screen）

## Trading OS 集成

基本面数据获取：

```bash
# AKShare 获取 A 股财务数据
python -c "
import akshare as ak
# ROE 数据
roe_df = ak.stock_roe_em(symbol='600000')
print(roe_df.tail(8))

# 机构持仓
inst_df = ak.stock_institute_hold(symbol='600000', quarter='20241')
print(inst_df.head())
"

# 北向资金持仓（外资）
python -c "
import akshare as ak
north_df = ak.stock_hsgt_individual_em(symbol='sh600000')
print(north_df.head())
"
```

CANSLIM 评分结果如果通过（≥5/7），直接调用 `elder-screen` 做技术面确认：
- 说"继续用三重滤网分析这只股票"
- elder-screen 会读取已有的技术面数据，不需要重复获取

两个维度综合后，再调用 `position-sizer` 计算仓位：
- CANSLIM 强烈关注（6-7/7）+ 技术面买入信号 → 正常仓位（2% 原则）
- CANSLIM 值得关注（4-5/7）+ 技术面买入信号 → 减半仓位（1% 原则）
- 两个维度冲突 → 不入场，等待方向一致
