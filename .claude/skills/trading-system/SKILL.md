# Trading System — AI 交易系统编排

整个交易系统的入口和调度中心。理解用户意图，选择合适的分析流程，
把各个专项 skill 的输出串联起来，形成完整的交易决策链。

---

## 系统架构（按用途分层）

```
trading-system（本 skill，编排层）
│
├── 【选股层】— 什么值得关注
│   ├── fundamental-research  深度基本面研究，估算内在价值（巴菲特/格雷厄姆框架）
│   ├── canslim-screen        快速基本面筛选，CANSLIM 七维度（欧奈尔框架）
│   └── signal-scanner        技术面批量扫描（三重滤网第一滤网）
│
├── 【分析层】— 深度分析单标的
│   └── elder-screen          三重滤网技术分析（埃尔德）
│
├── 【执行层】— 怎么买卖
│   ├── position-sizer    仓位计算（2%/6%原则）
│   ├── trade-executor    生成具体交易指令
│   └── position-monitor  持仓监控与止损追踪
│
└── 【复盘层】— 回顾改进
    ├── trading-journal   交易记录与绩效统计
    └── backtest-review   系统回测与健康评估
```

## 流程模式

根据用户意图，选择对应流程：

---

### 流程 A：基本面优先（中长线选股）

**触发词**："这只股票值得长期持有吗"、"帮我研究这家公司"、"这公司值多少钱"、
"深度分析"、"护城河是什么"、"估值合理吗"

**快速版**（用户只想初步筛选）：
```
1. [canslim-screen]          快速基本面评分（10 分钟）
2. [elder-screen]            技术面确认入场时机
3. [position-sizer]          仓位计算
4. [trade-executor]          生成指令
```

**深度版**（用户想认真研究一家公司）：
```
1. [canslim-screen]          快速初筛，是否值得深入（通过才继续）
2. [fundamental-research]    深度研究：护城河/财务/估值/逻辑止损条件（1 小时）
3. [elder-screen]            技术面确认入场时机
4. [position-sizer]          仓位计算
5. [trade-executor]          生成指令
```

**仓位规则**：
- fundamental-research 强烈关注 + 技术面买入 → 正常仓位（2% 原则）
- canslim ≥6/7 + 技术面买入（未做深度研究）→ 减半仓位（1% 原则）
- canslim 4-5/7 + 技术面买入 → 1/4 仓位，持续观察
- 两者冲突 → 不入场，明确告知冲突点

**止损模式**：基本面持仓用逻辑止损，不用价格止损。
持仓备注中记录 fundamental-research 输出的"逻辑止损条件"。

---

### 流程 B：技术面优先（短线机会）

**触发词**："扫描今天的机会"、"有什么突破信号"、"这个位置能进吗"、
"帮我做技术分析"、"今天哪些标的有信号"

```
1. [signal-scanner]   技术面批量扫描（输出信号清单）
2. [elder-screen]     三重滤网深度分析（TOP 信号逐一分析）
3. [canslim-screen]   基本面验证（这不是在炒梦吧）
4. [position-sizer]   仓位计算
5. [trade-executor]   生成指令
```

**说明**：技术面优先不代表忽略基本面。canslim-screen 在第 3 步做"否决验证"——
如果基本面极差（≤2/7），即使技术面有信号也不入场。

---

### 流程 C：综合分析（默认，适合大多数情况）

**触发词**："分析一下 600000"、"这只股票能买吗"、"帮我看看这个标的"

```
1. [canslim-screen]   基本面快速评分（重点看 C/A/M）
2. [elder-screen]     技术面分析（三重滤网）
3. [position-sizer]   仓位计算（如果两个维度都通过）
```

如果 canslim 评分 ≥ 5/7，询问用户是否需要深度研究：
"基本面初筛通过，要做深度研究（估值 + 护城河 + 逻辑止损条件）吗？约需 1 小时。"
- 是 → 插入 [fundamental-research]，完成后再做技术面
- 否 → 继续技术面分析，按减半仓位处理

---

### 流程 D：日常例行流程

**触发词**："跑今天的交易流程"、"开始今天的交易"、"运行交易系统"

```
1. [signal-scanner]   扫描候选标的池 → 输出信号清单
2. [elder-screen]     对高优标的深度分析
3. [canslim-screen]   对确认标的做基本面验证
4. [position-sizer]   计算仓位
5. [trade-executor]   生成今日交易指令
6. [position-monitor] 检查现有持仓，更新止损线
```

---

### 流程 E：持仓管理

**触发词**："检查我的持仓"、"有没有触及止损"、"帮我管理持仓"

```
1. [position-monitor] 检查所有持仓止损状态
2. [trade-executor]   如有需要，生成调仓指令
```

---

### 流程 F：复盘

**触发词**："记录这笔交易"、"更新交易记录"、"月度系统复盘"

```
1. [trading-journal]  记录新完成的交易
2. [backtest-review]  定期评估系统表现（每周/每月）
```

## 编排原则

**顺序依赖**：选股 → 分析 → 仓位 → 指令，顺序不能颠倒。

**两维度冲突处理**：
- 基本面好（≥5/7）但技术面差（周线下跌）→ 加入观察名单，等技术面改善
- 技术面好（买入信号）但基本面差（≤2/7）→ 不入场，可能是题材炒作
- 两者都好 → 正常入场
- 两者都差 → 不关注

**风险门控**（进入 trade-executor 之前必须确认）：
- 信号已通过两个维度验证
- 仓位已按 2% 原则计算
- 本月已用风险额度 + 本次风险 ≤ 账户的 6%
- 大盘方向（CANSLIM 的 M 维度）不是明确熊市

**透明输出**：每个步骤完成后向用户汇报，等待确认再继续。
不在用户不知情的情况下自动进入下一步（尤其是 trade-executor）。

## 会话状态

在一次完整的交易流程中，维护以下状态：

```
账户信息:
  - 账户总资金
  - 本月已亏损金额
  - 现有持仓列表（标的、入场价、止损价、仓位）

今日分析结果:
  - CANSLIM 评分（来自 canslim-screen）
  - 技术面信号（来自 signal-scanner / elder-screen）
  - 综合结论（两个维度的综合判断）
  - 待执行指令（来自 position-sizer + trade-executor）
```

## 启动方式

用户首次使用时，询问：
1. 账户规模（用于 2%/6% 计算）
2. 候选标的池（用于 signal-scanner 扫描）
3. 现有持仓情况（如有）
4. 偏好的分析重点（偏技术面 / 偏基本面 / 均衡）

这些信息可以保存在会话中，下次使用时直接确认是否有变化。

## 顶级交易员的共同原则（《市场奇才》施瓦格）

以下原则来自对 12 位顶级交易员的访谈，跨越技术分析、基本面分析、趋势跟踪等所有流派。
**这是超越方法论的共识，无论用哪种 skill 分析，都必须遵守。**

### 1. 止损是第一优先级

> "好交易的要素：(1) 止损，(2) 止损，(3) 还是止损。" —— Ed Seykota

- 止损必须实际挂单，而非只是"打算"出场（Michael Marcus）
- 止损设在技术关键位之外，而非由"我愿意亏多少"决定（Bruce Kovner）
- 当日组合亏损 1-2% 时，Paul Tudor Jones 可能立即清所有仓位

### 2. 让盈利奔跑，与止损同等重要

> "如果你不跟住赢家，就无法弥补输家的损失。" —— Michael Marcus

- 95% 的利润来自 5% 的交易，不能错过大行情（Richard Dennis）
- 最佳交易时仓位要大（三重确认时用 5-6 倍于常规的仓位），普通交易仓位要小

### 3. 仓位宁小勿大

> "把你认为应该持有的仓位至少减半。" —— Bruce Kovner

新手通常交易了 3-5 倍过大的仓位：
- 应有风险：单笔 1-2%（Kovner）；激进上限 5%（Marcus）
- 新手实际：单笔 5-10%（远超合理范围）
- 仓位过大 → 无意义波动 → 被震出好交易

### 4. 找到适合自己的方法，纪律性执行

施瓦格总结："**纪律（discipline）是被提及最频繁的词语。**"

- 系统化和主观交易各有顶级玩家，方法本身无对错，适合自己才是关键
- Ed Seykota："交易系统不需要改变，技巧是找到你与之相容的系统"
- 赚钱的欲望必须超过证明自己正确的欲望（Marty Schwartz）

### 5. 连续亏损时强制缩减仓位

| 情况 | 操作 |
|------|------|
| 连续亏损 | 不断缩减仓位直到重新回到正轨（Paul Tudor Jones） |
| 大亏损后 | 降至正常仓位的 1/5 到 1/10，等找回感觉（Marty Schwartz） |
| 极度失控 | 清空所有仓位，停止交易几天（施瓦格建议） |

**绝不因"想追平"而加大仓位。**

### 6. 大赢后特别小心

> "我最大的亏损总是跟在最大的盈利之后。" —— Marty Schwartz

- 连续盈利、感觉良好时，是最容易自满的时刻
- Paul Tudor Jones："我在觉得自己懂得什么的时候遭遇了最大的打击"

### 7. 耐心等待最佳时机

> "猎豹原则：等到绝对把握才出手，哪怕必须等一周。" —— Mark Weinstein

- 错过多数机会，只抓最确定的
- 扑克策略：好牌时重押，差牌时弃牌（Gary Bielfeldt）
- **有时"不做什么"比做什么更重要**

### 8. 亏损是游戏的一部分

- 成功交易员认为"亏损是可以接受的"（Van K. Tharp 研究）
- 区分"好交易"和"坏交易"应基于**过程**，而非结果
- "第一笔亏损是最好的亏损"——发现分析错误时立即出场（Jim Rogers）

### 9. 施瓦格最终总结的 9 条共同规律

1. 所有人都有强烈的成功欲望，克服了重大障碍
2. 坚信自己能长期盈利
3. **找到了适合自己的方法并坚守**（纪律是最高频词）
4. 非常认真对待交易，大量时间用于分析
5. **严格的风险控制是几乎所有人策略的关键要素**
6. 耐心等待正确的交易机会
7. 独立于人群行动
8. **所有人都理解亏损是游戏的一部分**
9. 热爱自己所做的事
