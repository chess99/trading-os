# Trading System — 体系导航

你好。你有三套独立的交易分析体系可以使用。

请告诉我你想做什么，我来推荐最合适的体系。

---

## 三套体系一览

### Elder 技术交易体系
**适合**：短中线技术交易，持仓周期天到周

- 核心方法：三重滤网（周线趋势 + 日线时机 + 精确入场）
- 止损方式：价格止损（2%/6% 原则）
- 数据需求：K 线行情数据
- **触发词**：说"**用 Elder 分析 [标的]**"或"**三重滤网分析**"

### CANSLIM 成长股体系
**适合**：中期成长股，持仓周到月

- 核心方法：基本面七维度筛选（欧奈尔）+ 技术面确认入场
- 止损方式：初期价格止损（7-8%），盈利后逻辑止损
- 数据需求：财务数据 + K 线行情
- **触发词**：说"**CANSLIM 分析 [标的]**"或"**成长股筛选**"

### Value Investing 价值投资体系
**适合**：长期价值投资，持仓月到年

- 核心方法：护城河分析 + DCF/SOTP 估值（巴菲特/格雷厄姆框架）
- 止损方式：纯逻辑止损（买入理由不再成立时卖出）
- 数据需求：财务报表、行业数据
- **触发词**：说"**估值分析 [标的]**"或"**价值投资分析**"

## 如何选择

| 你的问题 | 推荐体系 |
|---------|---------|
| "这个位置能进吗？" | Elder — 技术面时机判断 |
| "今天有什么突破信号？" | Elder — signal-scanner 扫描 |
| "这家公司成长性怎么样？" | CANSLIM — 基本面快速评分 |
| "这只股票值得长期关注吗？" | CANSLIM — 基本面+技术面综合 |
| "这公司值多少钱？" | Value Investing — 估值分析 |
| "这公司的护城河是什么？" | Value Investing — 深度研究 |
| "帮我检查持仓" | 取决于持仓属于哪套体系 |

## 重要原则（跨体系通用）

以下原则来自《市场奇才》施瓦格对 12 位顶级交易员的访谈，无论用哪套体系都适用：

1. **止损第一**：止损必须实际挂单，不能只是"打算"出场
2. **仓位宁小勿大**：把你认为应该持有的仓位至少减半（Kovner）
3. **连续亏损时强制缩减仓位**：不因"想追平"而加大仓位
4. **纪律性执行**：找到适合自己的方法后坚守，不随意切换

完整原则见 `references/principles.md`。
