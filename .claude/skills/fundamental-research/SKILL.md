# Fundamental Research — AI 深度基本面研究

AI 研究员模式，做的是传统基金公司研究员的工作：
读财报、理解业务、评估竞争壁垒、估算内在价值。

核心框架来自格雷厄姆《证券分析》、巴菲特历年股东信、芒格《穷查理宝典》。
但不照搬巴菲特——他的方法需要几十年的行业积累和极度集中的研究，
我们的目标是用 AI 提高信息处理密度，辅助判断，不是替代判断。

**与其他 skill 的分工：**
- `canslim-screen`：快速筛选，有没有成长特征，10 分钟
- `fundamental-research`：深度研究，这家公司值多少钱，1 小时
- `elder-screen`：技术面确认，现在能不能进场

---

## 研究框架（五个维度）

### 维度一：业务理解

**目标**：用一句话说清楚这家公司靠什么赚钱，为什么客户选它而不是竞争对手。

分析要点：
- 主营业务是什么，收入结构如何（产品/服务/地区分布）
- 客户是谁，为什么买这家的而不是别家的
- 商业模式：一次性收入还是重复性收入（订阅/耗材/服务更好）
- 行业处于什么阶段：成长期/成熟期/衰退期

**芒格检验**："如果我是竞争对手，我最怕这家公司的什么？"
如果答不上来，说明护城河不清晰。

---

### 维度二：护城河评估

**目标**：判断竞争优势是否持久，竞争对手能不能复制。

五种护城河（来自 Morningstar 框架，巴菲特认可）：

| 类型 | 特征 | A 股典型 |
|------|------|---------|
| 无形资产 | 品牌/专利/许可证 | 茅台（品牌）、恒瑞（专利） |
| 转换成本 | 客户换供应商代价高 | 金蝶/用友（ERP）、海康（安防） |
| 网络效应 | 用户越多越有价值 | 微信（已上市前）、交易所 |
| 成本优势 | 规模/流程/地理位置 | 宁德时代（规模）、海螺水泥（地理） |
| 有效规模 | 市场太小，不值得竞争 | 部分区域性公司 |

**护城河强度评级**：
- 宽护城河（Wide Moat）：竞争优势可持续 20 年以上
- 窄护城河（Narrow Moat）：竞争优势可持续 10 年左右
- 无护城河（No Moat）：竞争优势随时可能消失

**护城河侵蚀信号**（需要警惕）：
- 毛利率持续下滑
- 市场份额被新进入者蚕食
- 核心技术被替代
- 主要客户流失

---

### 维度三：财务健康度

**目标**：确认公司财务真实、可持续，没有隐藏风险。

#### 盈利质量

```
核心指标：
- 净利润 vs 经营现金流：两者应该基本匹配
  差距过大 = 应收账款堆积 or 利润注水
- 毛利率趋势：是否稳定或提升（下滑是护城河侵蚀信号）
- 净利率 vs 行业均值：高于行业均值说明有定价权
```

**格雷厄姆警示**：如果经营现金流长期低于净利润，这家公司的利润是纸面数字。

#### 资产负债健康度

```
- 资产负债率：制造业 < 60%，轻资产 < 40%（过高有财务危机风险）
- 有息负债 / EBITDA：< 3x 为健康，> 5x 需要警惕
- 商誉占比：商誉 / 总资产 > 30% 是风险信号（并购泡沫）
- 应收账款周转天数：是否在增加（客户议价能力下降信号）
```

#### 资本回报率

```
- ROE（净资产收益率）：> 15% 是优秀，连续 5 年以上更有说服力
- ROIC（投入资本回报率）：> WACC（加权平均资本成本）才是在创造价值
- 资本开支强度：高资本开支行业（重资产）护城河更难持久
```

#### 现金流分析

```
自由现金流 = 经营现金流 - 资本开支
- 自由现金流 / 净利润 > 80%：优质
- 持续负自由现金流：需要外部融资维持，风险较高
- 分红和回购历史：是否对股东友好
```

### 维度四：估值

**目标**：估算内在价值区间，判断当前价格是否提供了安全边际。

#### 方法一：DCF（折现现金流）— 最严谨

```
内在价值 = 未来自由现金流的折现值之和

关键假设：
- 未来 5-10 年的自由现金流增长率
- 永续增长率（通常 2-3%，不高于 GDP 增速）
- 折现率（WACC，通常 8-12%）

注意：DCF 对假设极其敏感，增长率差 1% 可能导致估值差 30%。
不要迷信精确数字，给出估值区间比单点估值更诚实。
```

#### 方法二：相对估值 — 最实用

```
PE（市盈率）：
- 与历史 PE 对比（当前 PE vs 过去 5 年平均 PE）
- 与行业 PE 对比（是否有溢价/折价，原因是什么）
- 注意：周期股用 PE 会误导，用 PB 或 EV/EBITDA 更合适

PEG = PE / 盈利增长率：
- < 1 通常被认为低估（但要确认增长质量）
- > 2 通常被认为高估

PB（市净率）：
- 适合银行、地产等重资产行业
- PB < 1 = 股价低于账面价值（但要确认资产质量）

EV/EBITDA：
- 排除资本结构和税率差异，跨公司比较更公平
- 制造业 < 10x 通常合理
```

#### 安全边际

格雷厄姆的核心原则：**以 6-7 折的价格买入内在价值。**

```
安全边际 = (内在价值 - 当前价格) / 内在价值

> 30%：较高安全边际，值得考虑
10-30%：一般安全边际
< 10%：安全边际不足，等待更好的价格
负值：当前价格高于内在价值，不买
```

A 股注意：政策风险、流动性溢价、散户情绪会导致价格长期偏离价值，
安全边际需要更高（> 40%）才能抵消这些不确定性。

---

### 维度五：风险清单

**目标**：列出可能让买入理由失效的具体风险。

必须检查的风险项：

```
行业风险：
□ 是否面临技术颠覆（电动车对传统燃油、AI 对传统软件）
□ 监管政策风险（教育、互联网、医疗等强监管行业）
□ 周期性风险（大宗商品、地产链、消费电子）

公司风险：
□ 大股东质押比例是否过高（> 50% 是警示）
□ 管理层是否有重大变动或负面新闻
□ 关联交易是否异常（利益输送信号）
□ 审计师意见是否有保留意见

财务风险：
□ 是否有大额商誉减值风险
□ 债务到期集中度（短期偿债压力）
□ 应收账款是否有坏账风险

估值风险：
□ 当前增速能否支撑当前估值
□ 市场预期是否过于乐观
```

## 逻辑止损 vs 价格止损

**这是本 skill 与技术分析 skill 最核心的区别。**

技术面持仓用价格止损（跌破某个价位就卖）。
基本面持仓用逻辑止损（买入理由不再成立就卖，和价格无关）。

**逻辑止损的触发条件（任何一条成立即卖出）：**

```
护城河层面：
- 核心竞争优势被竞争对手复制或替代
- 市场份额出现持续下滑（连续 2 个季度以上）
- 定价权丧失（毛利率持续下滑）

管理层层面：
- 核心管理层大规模离职
- 管理层出现重大诚信问题
- 资本配置明显恶化（大量低回报并购）

财务层面：
- 盈利质量持续恶化（现金流和利润持续背离）
- 负债大幅扩张且无清晰的还款来源
- 发现重大财务造假迹象

估值层面：
- 价格严重透支未来增长（安全边际变为负且幅度很大）
- 出现更好的投资机会（机会成本）
```

**不是逻辑止损的情况（不应该卖出）：**
- 股价下跌 20%、30%、50%（价格波动不是卖出理由）
- 市场情绪悲观，大家都在卖
- 短期业绩不达预期（但长期逻辑未变）

## 注意事项

- 财务数据可能有滞后，以最新季报为准
- A 股财务造假风险高于成熟市场，对"过于完美"的财报保持怀疑
- 这是辅助分析，不是投资建议；最终判断需要结合对行业的理解
- 深度研究耗时较长，建议只对已通过 canslim-screen 初筛的标的做深度研究
