# Elder 技术交易体系

基于亚历山大·埃尔德《以交易为生》的三重滤网系统。

**体系特征**：
- 持仓周期：天到周（短中线）
- 止损方式：价格止损，2%/6% 原则
- 决策依据：纯技术面（K 线、指标、价量关系）
- 基本面：仅作背景参考，不参与买卖决策

---

## 三重滤网体系架构

```
第一滤网（周线）— 只做顺势交易
  EMA 方向 + MACD 柱季节 → 确定多空方向

第二滤网（日线）— 等待回调入场机会
  随机指标 + 强力指数 → 找超卖/超买的入场时机

第三滤网（小时线/精确入场）— 精确执行
  买入止损单 / 卖出止损单 → 突破确认后入场
```

## 使用流程

### 流程 A：单标的深度分析

**触发词**："用 Elder 分析 600000"、"三重滤网分析 AAPL"

```
1. [elder-screen]     三重滤网完整分析
2. [position-sizer]   仓位计算（2%/6% 原则）
3. [trade-executor]   生成交易指令（如果信号通过）
```

### 流程 B：批量扫描 + 深度分析

**触发词**："今天有什么技术信号"、"扫描市场"、"跑今天的技术面流程"

```
1. [signal-scanner]   批量扫描候选标的池 → 输出信号清单
2. [elder-screen]     对 TOP 信号做三重滤网深度分析
3. [position-sizer]   仓位计算
4. [trade-executor]   生成今日交易指令
```

### 流程 C：持仓管理

**触发词**："检查 Elder 持仓"、"有没有触及止损"、"更新止损线"

```
1. [position-monitor] 检查所有持仓的价格止损状态
2. [trade-executor]   如有触及止损，生成平仓指令
```

## 体系规则

**止损规则**（不可妥协）：
- 每笔交易入场前必须确定止损位
- 止损触发时立即执行，不等待、不讨价还价
- 止损只能向盈利方向移动，绝不反向

**仓位规则**：
- 单笔风险 ≤ 月初账户的 2%
- 月度总风险（已亏 + 持仓敞口）≤ 月初账户的 6%
- 月度上限触发后，本月停止开新仓

**信号规则**：
- 必须三重滤网全部通过才入场
- 第一滤网（周线）确定方向，日线信号与周线方向一致才有效
- 不逆势交易，不在第一滤网反对时入场
