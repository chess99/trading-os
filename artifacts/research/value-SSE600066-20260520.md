## 【一句话总结】

宇通客车是中国客车行业的绝对龙头，零银行负债、ROE 35%+、**股息率6.3%（2025年报每股2元，当前价31.71）**，是优质商用车标的；但当前价格约等于 EPV（以 2025 年峰值利润计算），安全边际不足，属于**公允价值区间**，而非格雷厄姆意义上的"便宜货"。

## 【业务理解】

**主营**：大中型客车制造，覆盖公交、旅游、校车、海外出口四大市场。

**收入结构（2025）**：
- 国内公交/旅游/校车：营收约 300 亿（估）
- 海外出口（新能源为主，"一带一路"国家、拉美、中东）：营收约 100 亿+（快速增长）
- 年收入 414 亿，净利润 56.25 亿，净利率 13.6%（历史最高）

**客户为什么选它**：
- 国内：城市公交采购首选，大量地方国企客户，品牌积累 60+ 年
- 海外：中国最大客车出口商，新能源客车技术领先，性价比优于欧洲同类产品

**商业模式**：典型的重资产制造业，但有别于普通代工——宇通直销为主，掌握终端定价权。产品周期长（一辆客车使用 8-15 年），有一定的售后零件复购收入。

## 【护城河评估】

**来源分类**：规模经济 + 品牌/技术积累（窄护城河）

| 维度 | 评估 |
|------|------|
| 市场份额 | 国内客车市场 35-40%，远超第二名（宇通+中通+海格三家）|
| 技术壁垒 | 新能源客车研发领先，自研电池管理、整车控制系统 |
| 成本优势 | 规模最大，采购/制造成本低于竞争对手约 5-8% |
| 客户粘性 | 政府采购关系稳定，但基本靠招标，转换成本低 |
| 品牌 | 工具品（客车），品牌非消费品意义上的护城河 |

**EPV vs 资产再生产成本分析（格林沃尔德情形 C）**：
- 净有形资产（扣除现金和金融资产后）：约 70-80 亿
- EPV（2025 峰值利润 56 亿 ÷ 9%）：~622 亿
- EPV >> 资产价值，确认护城河存在
- 但关键问题：56 亿是否是可持续利润？

**护城河宽窄判断**：**窄（Narrow）**

- 正面证据：持续高市占率、零银行负债的强健财务、出口市场扩张
- 侵蚀风险：
  1. 新能源商用车竞争加剧（比亚迪 B 系商用、宇通以外其他友商）
  2. 客车市场天花板（不像乘用车市场庞大）
  3. 海外应收款风险（中东/非洲客户信用风险）

### 一美元原则检验

- 2021-2025 年累计净利润：129.13 亿
- 2021-2025 年累计分红（估）：约 100 亿（FY21: 11亿 + FY22: 11亿 + FY23: 33亿 + FY24: 33亿 + FY25: 55亿 ≈ 143亿，但分红率接近 100%，留存极少）
- **公司实质上把几乎所有利润作为分红返还给股东**，留存盈余很少。

一美元原则在此情形下结论：宇通是一家"价值分配型"而非"价值积累型"的成熟公司，不像伯克希尔那样内部再投资创造价值，而是靠高分红直接回报股东。对于价值投资者，这意味着——**必须把股息收益率计入回报预期**。

## 【2026Q1 下滑性质判断（核心问题验证）】

**问题**：2026Q1净利6.67亿 vs 2025Q1净利7.73亿，同比-13.6%，是否结构性恶化？

### 历年Q1对比（Q1是全年最弱季）

| 年份Q1 | 净利润 | 营收 | 净利率 |
|--------|--------|------|--------|
| 2026Q1 | 6.67亿 | 59.09亿 | 11.3% |
| 2025Q1 | 7.73亿 | 64.18亿 | 12.0% |
| 2024Q1 | 6.63亿 | 66.16亿 | 10.0% |
| 2023Q1 | 1.26亿 | 35.76亿 | 3.5% |
| 2022Q1 | -1.18亿 | 35.26亿 | 亏损 |
| 2021Q1 | -1.10亿 | 36.29亿 | 亏损 |
| 2020Q1 | -1.42亿 | 28.14亿 | 亏损 |

**判断：季节性+高基数效应，非结构性恶化。**

证据：
1. **2026Q1(6.67亿) > 2024Q1(6.63亿)**——与两年前同期对比是增长的，一季度绝对水平处于历史高位
2. 2025Q1（7.73亿）是异常高值，由2024年底政策激励+海外订单集中释放形成高基数
3. 2020-2022年Q1全部亏损，2024-2026年Q1全部盈利≥6.6亿，结构性改善已经确立
4. 净利率11.3%（2026Q1）接近正常水平，非毛利率恶化

**全年保守外推**：历史上Q1约占全年利润12-15%，则2026全年约44-56亿（6.67÷0.14≈47.6亿中性估计）。

## 【海外订单间接证据分析】

无法直接获取出口销量数据，但以下间接证据强烈支持海外扩张：

| 证据类型 | 数据 | 解读 |
|---------|------|------|
| 应收账款增加 | 2025年53.8亿 vs 2024年42.4亿（+11.4亿，+27%）| 海外账期60~120天 vs 国内30天，回款延迟是海外占比提升的直接信号 |
| 利润增速>营收增速 | 净利+35%，营收仅+11% | 高价值出口订单（新能源海外溢价）占比上升推动净利率从11.2%到13.6% |
| 毛利率扩张 | 24.1%（2025）vs 22.9%（2024）| 新能源出口车毛利率高于国内存量替换车型 |
| 分红力度 | 81%净利润分配（历史最高）| 管理层对未来现金流有信心，不需要为海外扩张保留大量资金 |

**风险量化**：
- 若2026年应收账款继续增至70亿（超过营收的12%），需重新评估坏账敞口
- 目前应收账款/营收 = 53.8/414 = 13%，处于历史正常区间（2020年是32%）
- 主要市场（中东、非洲、拉美）地缘风险是不可量化的尾部风险

## 【估值分析】

### 参数决策

| 参数 | 选择 | 理由 |
|------|------|------|
| 护城河 | Narrow（窄） | 市场份额领导 + NEV 技术，但无高转换成本/网络效应 |
| 资本成本 | 9% | 中风险商用车制造商，零债务降低一级，但行业周期性提高一级 |
| 可持续利润 | 45 亿（中性）/ 56 亿（乐观） | 扣非利润 45.8 亿（2025）作为中性基准，去除非经常性 |
| 增速假设 | 10-15%（5年） | 海外扩张保守估计，国内市场趋于稳定 |
| 终止 PE | 12-15x | 成熟制造业，非高成长，给相对保守 PE |

### 格林沃尔德三要素

**资产再生产成本**（净营运资产）：
- 净有形资产（存货 + 应收款 + 固定资产 - 经营负债）：约 70-90 亿
- 竞争对手进入需要：品牌积累（不可复制）+ 渠道（约需 10-15 年）
- 估计资产再生产成本：**约 150-200 亿**

**EPV（无成长假设）**：
- 扣非可持续利润：45 亿 ÷ 9% = **500 亿 = 22.6 元/股**
- 含 2025 峰值全利润：56 亿 ÷ 8% = **700 亿 = 31.6 元/股**
- EPV >> 资产再生产成本，确认护城河存在（格林沃尔德情形 C）

**成长价值**：
- 当前 ROIC 远高于资本成本（护城河确认）
- 成长可以创造价值，但成长空间有限（客车市场非大众市场）
- 海外扩张是核心增量，估计可支撑 5-7 年 10-15% 利润增长

### DCF 估值

基于扣非利润 45 亿，折现率 10%，5 年成长期：

```
增速 15%，终止 PE 15x → 37.0 元/股（17% 上行空间）
增速 10%，终止 PE 12x → 23.7 元/股（25% 下行风险）
增速 20%，终止 PE 15x → 45.7 元/股（44% 上行空间）
```

## 【风险清单】

1. **2025 利润可能是阶段性峰值**：净利率从历史 6-8% 提升至 13.6%，驱动因素（出口高增、政府补贴约 10 亿）可能不可持续。若回落至 10%，对应利润约 41 亿（接近 2024 年），支撑现价问题不大；若回落至 7%，仅约 29 亿，则当前估值偏贵。

2. **海外应收款信用风险**：出口高增带来大量应收票据（2025 年应收账款 55 亿+）。新兴市场客户（中东、非洲、拉美）支付周期长，地缘风险高。

3. **国内市场天花板**：城市公交新能源替换接近完成，未来国内增量更多来自更换周期（8-12 年），系统性成长有限。

4. **竞争加剧**：比亚迪等乘用车巨头通过商用车业务进入，中长期可能影响市场份额和利润率。

5. **政策依赖**：新能源补贴退坡后，成本竞争加剧；出口退税政策变化风险。
