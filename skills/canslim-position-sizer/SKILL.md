# CANSLIM Position Sizer — 成长股仓位计算

与 Elder 体系的区别：Elder 用 ATR 止损（基于波动率），CANSLIM 用固定百分比止损（欧奈尔实测 7-8%）。

---

## 核心规则

### 欧奈尔止损规则

**初始止损**：任何新建仓位，若价格从入场价下跌 7-8%，立即止损出场。无例外。

```
止损价 = 入场价 × (1 - 止损比例)
止损比例：7% 或 8%（根据股票波动性选择）
```

- 波动性低的龙头股（如消费白马）：7% 止损
- 波动性高的成长股（如科技股）：8% 止损

### 仓位计算

```
最大风险金额 = 账户总资金 × 2%（单笔风险上限，与 Elder 体系一致）
每股风险 = 入场价 - 止损价 = 入场价 × 止损比例
最大仓位 = 最大风险金额 / 每股风险
```

**示例**：
- 账户 50 万，2% 风险上限 = 1 万
- 入场价 100 元，8% 止损 = 止损价 92 元，每股风险 8 元
- 最大仓位 = 10,000 / 8 = 1,250 股

### 仓位分级（基于 CANSLIM 评分）

| CANSLIM 评分 | 建议仓位比例 |
|-------------|------------|
| 7/7（全部通过） | 满额（2% 风险） |
| 5-6/7 | 半额（1% 风险） |
| 4/7 | 1/4 额（0.5% 风险），谨慎观察 |
| < 4/7 | 不建议入场 |

## 输出格式

```

  账户总资金：[X] 元
  单笔风险上限（2%）：[X] 元

本次交易：
  入场价：[X]
  止损价：[X]（入场价的 [7/8]% 以下）
  每股风险：[X] 元
  CANSLIM 评分：[X]/7

计算结果：
  满额仓位：[X] 股
  建议仓位（按评分分级）：[X] 股（[X]% 风险）
  实际风险金额：[X] 元

→ 下一步：运行 canslim/trade-executor 生成交易指令
```
