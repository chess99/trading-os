# 基金管理AI系统文档

## 系统概述

这是一个基于Claude Code构建的自主基金管理系统。您作为董事长负责战略决策，AI作为基金经理负责日常分析和执行。

## 快速开始

```bash
# 测试系统
python test_agent_system.py

# 日常分析
python -m trading_os agent daily

# 获取投资建议
python -m trading_os agent recommend

# 风险评估
python -m trading_os agent risk

# 董事会报告
python -m trading_os agent board-report
```

## 架构设计

### 当前实现
- **单Agent模式**: 基金经理AI集成多种技能
- **技能组合**: 市场分析 + 风险评估 + 决策生成
- **模块化设计**: 核心接口 + 技能模块 + 角色实现

### 与Claude Code集成
- 使用现有的trading-os基础设施
- 集成到CLI命令中
- 保持与数据湖、回测系统的兼容

## 使用指南

### 作为董事长，您的职责
1. **战略方向**: 制定总体投资策略和风险偏好
2. **重大决策**: 对AI建议进行最终拍板
3. **资源授权**: 提供必要的数据和执行权限
4. **监督指导**: 定期review AI的决策和表现

### AI基金经理的能力
1. **市场分析**: 趋势识别、行业轮动、技术指标
2. **风险管理**: 组合风险评估、市场风险监控
3. **投资决策**: 基于分析生成具体投资建议
4. **报告生成**: 自动生成分析报告和董事会汇报
