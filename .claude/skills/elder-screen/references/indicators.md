# 技术指标计算参考

## EMA（指数移动平均）

```
EMA = 当日收盘价 × K + 前日EMA × (1-K)
K = 2 / (N+1)，N 为天数
```

常用参数：
- 短期：13日（周线）/ 11日（日线）
- 长期：26日（周线）/ 22日（日线）
- 价值区间 = 两条EMA之间的空间

## MACD

```
MACD线 = 12日EMA - 26日EMA
信号线 = MACD线的9日EMA
MACD柱 = MACD线 - 信号线
```

柱状线斜率方向比正负值更重要：
- 柱状线上升（后柱 > 前柱）→ 多头力量增强
- 柱状线下降（后柱 < 前柱）→ 空头力量增强

## ATR（平均真实波幅）

```
真实波幅 TR = max(当日高-当日低, |当日高-昨收|, |当日低-昨收|)
ATR = TR 的 13日均值
```

用途：
- 止损距离至少 1倍ATR
- 盈利目标：1/2/3倍ATR处分批兑现

## 随机指标（KDJ）

```
%K = (今收 - N日最低) / (N日最高 - N日最低) × 100
%D = %K 的3日移动平均
```

标准参数：N=5，慢速随机（再平滑一次）
参考线：上80%（超买），下20%（超卖）

## 强力指数

```
强力指数 = 今日成交量 × (今收 - 昨收)
2日EMA：短期入场信号
13日EMA：中期趋势确认
```

## 趋向系统（ADX）

- +DI > -DI → 多头主导
- -DI > +DI → 空头主导
- ADX 从趋向线下方低点回升4个点 → 新趋势开始的强信号
- ADX > 两条趋向线后掉头 → 趋势过热，兑现利润
