# 子 Skill 接口规范

各子 skill 之间传递数据的标准格式，确保编排层能正确衔接。

## signal-scanner 输出

```
信号清单:
  - 标的: AAPL
    方向: 做多
    信号强度: 强/中/弱
    周线季节: 春（MACD柱0值以下向上）
    日线状态: 随机指标超卖，2日强力指数转负
    备注: 出现向下袋鼠尾，支撑位附近

  - 标的: 黄金期货(GC)
    方向: 观望
    信号强度: 弱
    ...
```

## elder-screen 输出

```
深度分析结果:
  标的: AAPL
  第一滤网: 周线EMA上升，MACD柱春季 → 只做多
  第二滤网: 日线随机指标28（超卖），2日强力指数-1200（负值）
  背离: 无
  第三滤网: 建议入场价 182.50（前日高点上方）
  止损价: 179.80（支撑位下方，距入场1.5%）
  目标价1: 185.20（1倍ATR）
  目标价2: 187.90（2倍ATR）
  风险收益比: 2.8:1
```

## position-sizer 输出

```
仓位计算结果:
  标的: AAPL
  账户规模: 500,000
  单笔最大风险(2%): 10,000
  入场价: 182.50
  止损价: 179.80
  每股风险: 2.70
  建议仓位: 3,703 股（约 675,000，需确认是否超过账户比例上限）
  本月已用风险: 3,200（0.64%）
  本次风险: 10,000（2%）
  累计风险: 13,200（2.64%）← 未超过6%，可以交易
```

## trade-executor 输出

```
交易指令清单:
  指令1（新开仓）:
    操作: 买入
    标的: AAPL
    数量: 3,703 股
    类型: 限价单
    价格: 182.50
    止损单: 179.75（低于整数位几美分）
    备注: 三重滤网确认，风险收益比2.8:1

  指令2（调整止损）:
    操作: 修改止损
    标的: NVDA（现有持仓）
    原止损: 420.00
    新止损: 445.00（移至盈亏平衡点上方）
    原因: 价格已上涨超过1倍ATR
```

## trading-journal 输入

```
交易记录（每笔完成的交易）:
  日期: 2024-01-15
  标的: AAPL
  方向: 做多
  入场价: 182.50
  出场价: 186.20
  仓位: 3,703 股
  盈亏: +$13,330（+2.0%）
  持仓天数: 3
  使用系统: 三重滤网（春季买入）
  入场理由: 周线春季+日线超卖+支撑位
  出场理由: 接近1倍ATR目标，主动止盈
  绩效评级: A（赚取通道高度30%以上）
  复盘备注: 信号清晰，执行到位
```
