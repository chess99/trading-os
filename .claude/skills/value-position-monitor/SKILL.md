# Value Investing Position Monitor — 逻辑止损监控

价值投资体系没有价格止损。卖出的唯一理由是：**买入时的逻辑不再成立**。

价格下跌本身不是卖出理由。如果公司基本面没有变化，价格下跌反而可能是加仓机会。

---

## 逻辑止损检查清单

每季度财报后，或发生重大事件时，对每个持仓逐一检查：

### 护城河完整性
- [ ] 核心竞争优势是否仍然存在（品牌力、网络效应、转换成本、成本优势）
- [ ] 市场份额是否被蚕食（竞争对手是否有突破性进展）
- [ ] 定价权是否仍然存在（是否还能提价而不失去客户）

### 财务质量
- [ ] ROE 是否持续高于 15%（连续两季度下滑则警惕）
- [ ] 自由现金流是否健康（净利润与经营现金流是否匹配）
- [ ] 负债率是否恶化（特别是有息负债快速增加）
- [ ] 毛利率是否持续下滑（成本压力或定价权丧失的信号）

### 管理层
- [ ] 管理层是否仍然诚信（有无财务造假迹象、关联交易异常）
- [ ] 资本配置是否合理（是否在低回报项目上大量投资）
- [ ] 是否有重大人事变动（创始人/核心管理层离职）

### 买入假设
- [ ] 买入时的核心论点是否仍然成立
- [ ] 买入时预期的催化剂是否已经发生或已经失效
- [ ] 行业格局是否发生根本性变化

---

## 卖出触发条件

**立即考虑卖出**（任一条触发）：
- 护城河出现明显裂缝（竞争对手大幅抢占市场份额）
- 管理层诚信出现问题（财务造假迹象）
- 核心假设被证伪（买入时的关键逻辑已经不成立）

**减仓观察**（需要持续跟踪）：
- ROE 连续两季度下滑超过 20%
- 自由现金流与净利润背离加剧
- 行业竞争格局恶化但护城河仍在

**继续持有**（这些不是卖出理由）：
- 价格下跌（只要基本面没变）
- 短期业绩低于预期（如果是一次性因素）
- 市场整体下跌
