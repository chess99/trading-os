# CANSLIM Trade Executor — 成长股交易指令

将 CANSLIM 分析结果（入场价、止损价、仓位）转化为可执行的交易指令。

---

## 指令类型

### 新建仓位

```
指令1（买入）：
  操作：买入
  标的：[代码 名称]
  数量：[X] 股
  类型：限价单（枢纽点突破买入）或 买入止损单（突破确认）
  价格：[入场价]
  有效期：当日有效（枢纽点突破机会稍纵即逝）

指令2（初始止损）：
  操作：卖出止损单
  标的：[代码 名称]
  数量：[X] 股（全部仓位）
  触发价：[止损价]（入场价的 7-8% 以下）
  类型：止损市价单
  备注：CANSLIM 初期止损，触发后立即以市价成交
```

### 止损上移（浮盈达到 20% 时）

```
指令：修改止损单
  标的：[代码 名称]
  原止损价：[X]
  新止损价：[入场价]（移至成本线）
  备注：浮盈已达 20%，切换为成本线保护；后续改用逻辑止损
```

---

## 执行注意事项

**枢纽点买入的时间敏感性**：
- 突破枢纽点后 5% 以内：可以追入
- 突破枢纽点超过 5%：等待回调，不追高
- 突破当日成交量须比 50 日均量放大 ≥ 50%，否则突破可信度低

**止损单的重要性**：
- 买入后立即挂止损单，不要"等等看再挂"
- 止损单是保护本金的最后防线，欧奈尔原则：无论如何不让单笔亏损超过 8%

---

```
## 执行指令

指令1：买入
  标的：[代码] [名称]
  数量：[X] 股

  有效期：当日

指令2：初始止损
  触发价：[X]（入场价 [X]% 以下）
  数量：[X] 股（全部）
  类型：止损市价单

## 风险确认
  仓位金额：[X] 元
  最大亏损：[X] 元（[X]%）
  ✓ 符合 2% 单笔风险原则

[请确认后执行，或输入"驱动 paper 命令"自动执行]
```
