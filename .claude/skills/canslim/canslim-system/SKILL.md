# CANSLIM 成长股体系

基于威廉·欧奈尔《笑傲股市》（How to Make Money in Stocks）的完整成长股投资系统。

**体系特征**：
- 持仓周期：周到月（中期）
- 止损方式：初期价格止损（7-8%），盈利后逻辑止损
- 选股逻辑：基本面七维度筛选出优质成长股，技术面确认最佳入场时机
- 核心洞见：最大涨幅的股票在起涨前都有共同的基本面特征；技术面告诉你何时买，基本面告诉你该买什么

---

## 体系架构

```
第一步：基本面筛选（canslim-screen）
  CANSLIM 七维度评分 → 确认公司成长质量

第二步：技术面确认（elder-confirm）
  日线动量 + 精确入场 → 确认当前是否为最佳买点
  （只用第二、三滤网；基本面研究已替代第一滤网的趋势过滤作用）

第三步：仓位计算（canslim/position-sizer）
  欧奈尔 7-8% 初始止损规则 → 计算仓位大小

第四步：交易指令（canslim/trade-executor）
  生成入场单、初始止损单

第五步：持仓管理（canslim/position-monitor）
  初期：价格止损（7-8%）
  盈利后：切换逻辑止损（基本面恶化时卖出）
```

---

## 使用流程

### 流程 A：单标的完整分析

**触发词**："CANSLIM 分析 600000"、"帮我做 CANSLIM 评分"

```
1. [canslim-screen]          CANSLIM 七维度基本面评分
   → 评分 ≥ 5/7：继续技术面确认
   → 评分 < 4/7：不建议入场，说明原因

2. [elder-confirm]           技术面确认（第二+三滤网）
   → 通过：计算仓位
   → 未通过：加入观察名单，等待时机

3. [canslim/position-sizer]  仓位计算（7-8% 止损规则）

4. [canslim/trade-executor]  生成交易指令
```

### 流程 B：持仓管理

**触发词**："检查 CANSLIM 持仓"、"帮我更新逻辑止损"、"持仓基本面有没有变化"

```
1. [canslim/position-monitor] 检查每个持仓的止损状态
   → 初期持仓：检查价格是否跌破 7-8% 止损线
   → 盈利持仓：检查逻辑止损条件（基本面恶化信号）
```

---

## 两种止损的切换时机

| 持仓阶段 | 止损类型 | 触发条件 |
|---------|---------|---------|
| 入场后初期（浮盈 < 20%） | 价格止损 | 跌破入场价 7-8% 立即平仓 |
| 盈利阶段（浮盈 ≥ 20%） | 逻辑止损 | 基本面恶化：EPS 增速连续两季下滑 2/3、失去行业龙头地位、机构持仓大幅减少 |
| 任何阶段 | 欧奈尔卖出规则 | 股价从最高点回落 8% = 强制止盈/止损 |

---

## 体系边界

**CANSLIM 体系只做**：
- 有真实业绩支撑的成长股（EPS 持续高增长）
- 行业龙头（相对强度排名前 10%）
- 大盘方向不是明确熊市时（M 维度）

**CANSLIM 体系不做**：
- 纯技术面信号、没有基本面支撑的交易（→ 用 Elder 体系）
- 低估值、低增长的价值股（→ 用 Value Investing 体系）
- 逆势做空（欧奈尔只做多）
